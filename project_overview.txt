# TASK.TXT

# Project Name

Personal AI Commerce Assistant

---

# Project Overview

Build a next-generation AI-powered commerce assistant that transforms shopping from a traditional search-based experience into a conversational, personalized, proactive experience.

Traditional e-commerce platforms require users to:

* Search products
* Compare products
* Read reviews
* Manage carts
* Repeat monthly purchases

This project aims to eliminate these friction points.

Instead of asking:

"What product do you want?"

The assistant asks:

"What do you need?"

The system should understand:

* User preferences
* Family requirements
* Shopping habits
* Health preferences
* Calendar events
* Weather conditions
* Purchase history
* Budget constraints
* Lifestyle information

and proactively help users discover, plan, and purchase products.

---

# Primary Goal

Create an AI assistant capable of:

* Understanding user intent
* Recommending products
* Creating complete shopping carts
* Explaining recommendations
* Managing addresses
* Managing carts
* Performing checkout through chat
* Learning user behavior over time

The experience should feel like:

ChatGPT + Amazon + Personal Household Assistant

---

# Core Product Philosophy

Users should not need to search for products.

Users should simply describe:

* Problems
* Goals
* Situations
* Events
* Requirements

Examples:

"I need groceries for a family of four."

"Planning a birthday party."

"Need healthy breakfast options."

"Going trekking this weekend."

"My child starts school next week."

The assistant should convert these situations into actionable shopping recommendations and shopping carts.
